Happy Greens

Happy Greens is a recipe app for those living a plant-based or vegetarian lifestyle that want more family-friendly comfort food. ​It also provides meat-eaters with an easier transition to a plant-based lifestyle since they can access more comforting and traditional meals that they are used to.

My Process

User Research

The Research Learning Spiral (RLS) was chosen as the research method to help gather knowledge about various users and their needs. Three users were recruited; a 23 year-old female, a 28 year-old male, and a 57 year-old female.

Patterns that emerged from the answers were that most participants mentioned wanting to eat healthier and enjoyed finding recipes online.

User Personas

Three user personas were then created based off the insights gathered from the user testing.

User Flows

A user flow diagram was created to represent all of the steps a user would take.

Low and Mid Fidelity Wireframes

Low-fidelity wireframes were then created using the Crazy 8s method to generate ideas for the user flows. These sketches were then turned into digital prototypes using InVision for usability testing. Later mid-fidelity wireframes were created digitally using Adobe XD using the feedback from the usability testing.

Style Guide

A style guide was created to set guidelines for color, typography, and button styles to ensure uniformity in design.

Mockups and Responsive Design

High-fidelity screens in color were then created, adding the colors and typography stated in the style guide. Responsive designs for the home screen were created for mobile, tablet and desktop so that the product can adapt easily to a variety of screen sizes.

Conclusion

Throughout the process of creating Happy Greens, I tried to make it different from other recipe apps. The design is meant to convey a sense of comfort to go along with the theme of comfort family-friendly eating. This was done by looking at farmhouse-style interior décor for inspiration. This project helped me develop my user testing and research skills since I conducted initial user research, developed user personas, created user flows, and conducted usability testing.
